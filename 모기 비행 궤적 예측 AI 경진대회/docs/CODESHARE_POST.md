# 데이콘 코드 공유 게시글 (본문 초안)

> 실제 데이콘 수상작 코드공유 컨벤션에 맞춤: **제목=순위 명시**, **본문은 짧고 겸손한 존댓말**(상세 방법론은 첨부 PDF), **첨부 = PDF 발표자료 + 코드 노트북**. 아래를 그대로 붙여넣고 PDF·노트북 2개를 첨부하세요.

---

**제목**

```
[Private 2위 0.703151] 직교 메커니즘 앙상블 — 코드 + 발표자료 공유
```

**본문**

안녕하세요. 모기 비행 궤적 예측 대회에서 운 좋게 **Private 2위(0.703151)** 를 기록했습니다. 솔루션 발표자료(PDF)와 통합 코드(노트북)를 공유합니다.

이 대회 지표(R-Hit@1cm)가 **1cm 경계에 걸친 샘플을 넘기느냐**로 점수가 갈리는 이진 hit 지표라, 단일 모델 정확도보다 **서로 직교한 예측 메커니즘의 다양성**이 핵심이라고 보고 접근했습니다. 같은 Kalman 잔차 base 위의 모델들이 상관 ~0.99로 묶여 LB 0.6888에서 막혔는데, **Neural ODE → Frenet 3D-프레임 → 회전물리(CREE)** 처럼 근본적으로 다른 base를 단계적으로 더하며 plateau를 넘었습니다. 특히 **OOF가 더 낮아도 더 직교한 멤버를 수동으로 더 섞을수록(over-conversion) LB가 올라간다**는 점이 결정적이었습니다. 단계별 점수·실패 사례까지 자세한 내용은 첨부 PDF에 정리했습니다.

- **코드**: 첨부 노트북 1개. 마지막 셀(§7)로 최종 제출이 **외부 의존 없이 1초 만에 재현**됩니다(원본과 오차 < 0.001mm).
- **규칙 준수**: 회전물리 멤버(내부 코드네임 CREE — **동명의 참가자와 무관**)는 대회 기간 중 코드 공유 게시판에 공개됐던 회전물리 baseline의 **모델 구조를 참고**(규칙 8조 B항 공개 코드 허용)했고, **가중치 차용 없이 train만으로 from-scratch 학습**했습니다. test 데이터 학습·외부 데이터·원격 API 사용 없으며 시드 고정 재현 가능합니다.
