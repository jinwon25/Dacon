# RE:TACTIC 시연 영상 대본

목표 길이: 2분 30초. 화면 녹화는 데스크톱 1440×900 이상, 브라우저 확대 100%를 권장합니다.

## 0:00–0:15 문제와 콘셉트

- 독립 홈의 “경기를 읽고, 전술로 답하다”를 보여준 뒤 세 경기 선택 화면으로 이동합니다.
- 내레이션: “RE:TACTIC은 실제 월드컵의 결정적 순간을 데이터로 진단하고, 내가 만든 개입안의 이점과 위험을 비교하는 코칭 워크스페이스입니다.”

## 0:15–0:45 2026 공식 경기 사후 리뷰

- `2026 남아프리카공화국–대한민국`을 선택하고 64분 추격 미션을 시작합니다.
- FIFA 전체 경기 보고서의 점유 60.9%, 패스 성공률 92%, 슈팅 8회를 짚습니다.
- 경기 국면 카드에서 전진량과 마지막 행동의 간극을 확인합니다.
- 패스맵에서 이기혁을 눌러 직접 연결만 강조합니다.
- 전체 경기 공식 수치와 64분 사후 재구성을 화면에서 구분한다는 점을 말합니다.

## 0:45–1:25 전술 설계

- 3-4-3에서 선수 한 명을 드래그하고 이강인의 역할과 압박·폭·템포·위험 감수를 조절합니다.
- 김민재를 박진섭으로 교체하고 오른쪽 비교 지표가 즉시 바뀌는 장면을 보여줍니다.
- 결과 검토로 이동합니다.

## 1:25–1:50 두 데이터 경험 비교

- 전술 개입 메모의 기준선 변화, 이점·리스크·운영 조건을 짚습니다.
- 한국–포르투갈 브리핑으로 잠시 전환해 실제 이벤트 좌표의 공격 진입·슈팅과 볼 흐름 재생을 보여줍니다.
- “공개 좌표가 있을 때만 공간 장면을 재생한다”는 원칙을 강조합니다.

## 1:50–2:25 범용 전술 스튜디오

- 상단에서 범용 전술 스튜디오로 전환합니다.
- 팀·상대·목표를 입력하고 상대 4-3-3 오버레이 선수를 움직입니다.
- 프랑스 역습 템플릿을 적용하고 패스·침투 화살표를 하나 추가합니다.
- 현재 배치를 저장한 뒤 `다음 장면`을 추가해 선수 위치를 바꾸고 전체 재생합니다.
- 저장된 기존 위치가 점선 마커로 남아 Before/After를 보여주는 장면을 담습니다.
- A안 저장 후 포메이션·위험 감수를 바꿔 B안에 저장하고 차이를 비교합니다.
- 분석 탭에서 코치 제안을 미리 보고 적용해 지표가 바뀌는 장면을 보여줍니다.
- 공간 지배도를 켜고 선수를 움직일 때 영역이 즉시 바뀌는 장면을 보여줍니다.
- PNG 저장을 눌러 코칭 미팅용 16:9 전술안이 생성되는 장면을 보여줍니다.

## 2:25–2:30 마무리

- 내레이션: “게임 결과를 맞히는 대신, 실제 관측값 위에서 더 나은 전술 대화를 만들었습니다. RE:TACTIC, 그때 내가 감독이었다면.”
