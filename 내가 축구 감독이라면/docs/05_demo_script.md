## 0:00–0:15 문제와 콘셉트

- 시작 화면에서 한국–포르투갈 65분, 1–1 상황을 보여줍니다.
- 내레이션: “RE:TACTIC은 실제 월드컵의 결정적 순간을 데이터로 진단하고, 내가 만든 개입안의 이점과 위험을 비교하는 코칭 워크스페이스입니다.”

## 0:15–0:45 데이터 브리핑

- `실제 경기로 익히기`를 누릅니다.
- 45–64분 패스 성공률, 공격 지역 진입, 압박을 짚습니다.
- 공간 증거 탭에서 한국과 포르투갈의 실제 진입 패스·슈팅 위치를 비교합니다.
- “관측 데이터”와 “코치의 해석”이 함께 제공된다는 점을 강조합니다.

## 0:45–1:25 전술 설계

- 4-3-3에서 3-4-3으로 바꾸고 선수 한 명을 드래그합니다.
- 손흥민 역할과 압박·폭·템포·위험 감수를 조절합니다.
- 이재성을 황희찬으로 교체하고 오른쪽 비교 지표가 즉시 바뀌는 장면을 보여줍니다.
- 결과 검토로 이동합니다.

## 1:25–1:50 근거 있는 검토

- 실제 55분 조규성→손흥민 슈팅 장면을 재생합니다.
- 제안 개입 탭으로 바꾸어 실제 좌표와 설명용 움직임이 구분됨을 보여줍니다.
- 전술 개입 메모의 이점·리스크·운영 조건을 짚습니다.

## 1:50–2:25 범용 전술 스튜디오

- 상단에서 범용 전술 스튜디오로 전환합니다.
- 팀·상대·목표를 입력하고 상대 4-3-3 오버레이 선수를 움직입니다.
- A안 저장 후 포메이션·위험 감수를 바꿔 B안에 저장하고 차이를 비교합니다.
- PNG 저장을 눌러 코칭 미팅용 16:9 전술안이 생성되는 장면을 보여줍니다.
