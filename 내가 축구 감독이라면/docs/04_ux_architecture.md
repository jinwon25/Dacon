# UX 정보 구조와 초보자 설계

## 1. 제품 구조

RE:TACTIC은 숙련도와 사용 목적이 다른 두 진입점을 제공합니다.

### 실제 경기 분석

- 대상: 처음 방문한 축구 팬, 심사자, 데이터 분석 입문자
- 목적: 실제 데이터로 문제를 이해하고 전술 조작의 의미를 짧게 학습
- 흐름: 시작 → 데이터 진단 → 전술 설계 → 의사결정 검토
- 권장 진입점으로 표시

### 범용 전술 스튜디오

- 대상: 코치, 분석관, 전술 보드를 이미 이해하는 사용자
- 목적: 특정 국가나 경기 시나리오에 제한되지 않고 자신의 전술안을 작성
- 흐름: 경기 정보 → 선수 배치 → 팀 지시 → 구조 점검·저장
- 브라우저에 자동 저장

## 2. 핵심 UX 원칙

### 선택보다 추천을 먼저

첫 방문자에게 두 모드를 같은 무게로 던지지 않습니다. `실제 경기로 익히기`를 기본 행동으로 강조하고 범용 스튜디오는 보조 행동으로 둡니다.

### 설명보다 즉시 조작

긴 튜토리얼 대신 피치 중앙 코치마크로 첫 행동을 알려줍니다. 선수를 한 번 움직이면 안내가 사라집니다.

### 항상 다음 행동이 보이게

화면 위에 4단계 작업 순서를 고정 표시합니다. 각 패널의 번호도 이 순서와 일치합니다.

### 필요한 정보만 단계적으로

우측 패널은 `선수·분석·전술안` 탭 중 하나만 보여줍니다. 선수를 누르면 선수 편집, 경기장 빈 공간을 누르면 구조 분석으로 자동 전환해 Football Manager식 깊이는 유지하되 정보 밀도는 낮춥니다.

### 전문 용어를 일상어와 함께

`파이널서드` 대신 `공격 지역`, `프로젝션` 대신 `시나리오 비교`, `매치 엔진` 대신 `구조 점검`을 사용합니다.

### 실제 값과 모델 값을 분리

관측된 경기 기록, 사용자가 입력한 전술, 휴리스틱 비교 결과를 다른 문구와 위치로 구분합니다. 구조 점수는 실제 승률처럼 표현하지 않습니다.

### 되돌릴 수 있는 조작

포메이션 프리셋, 배치 원위치, 전체 초기화를 제공합니다. 초기화는 확인 후 실행하고 일반 편집 내용은 자동 저장합니다.

## 3. 범용 스튜디오 기능

- 팀·상대·경기 단계·시간·스코어·목표 편집
- 포메이션 변경 및 선수 자유 배치
- 선수명·등번호·역할 편집
- 벤치 교체
- 압박·공격 폭·템포·위험 감수 조정
- 팀 폭, 수비 라인, 전방 지원, 전환 위험 진단
- 현재 목표에 맞는 코칭 노트
- 상대 포메이션 오버레이와 상대 선수 자유 배치
- 대한민국 4-2-3-1, 아르헨티나 4-3-3, 프랑스 역습, 코너킥 템플릿
- 패스·침투 도구와 최대 5개 장면 타임라인
- 장면 순차 재생과 저장 장면 기준 Before/After 점선 위치
- 전술안 A/B 저장·불러오기·차이 비교
- 배치 실행 취소와 로컬 자동 저장
- 16:9 PNG와 인쇄·PDF 출력
- 규칙 기반 코치 제안의 위치 미리보기와 원클릭 적용

## 4. 반응형 우선순위

### 데스크톱

경기 정보와 팀 지시를 왼쪽, 전술판을 중앙, 선수 편집과 구조 점검을 오른쪽에 배치합니다. 코칭 스태프가 한 화면에서 입력과 결과를 비교하는 데 초점을 둡니다.

### 모바일

`경기 정보 → 전술판 → 팀 지시 → 선수 편집·검토` 순서로 세로 배치합니다. 첫 조작까지의 스크롤을 줄이고 포메이션 버튼은 2×2로 표시합니다.

## 5. 후속 개선 후보

1. 사용자 JSON 경기 데이터 가져오기
2. 키보드만으로 선수 이동 가능한 접근성 모드
3. 전술 시퀀스의 다중 프레임 편집
4. 팀별 템플릿 라이브러리
5. 오프라인 사용을 위한 PWA 설치 옵션

현재 해커톤 MVP에서는 첫 사용의 명확성과 오류 없는 핵심 조작을 우선합니다.
